# Brand System: Dynamous

## Overview
Dynamous is a collaborative platform for teams to build faster. We combine elegance and power — a tool that feels effortless but delivers results.

## Design Philosophy
- **Form follows function.** Every visual choice has a reason. No decoration for its own sake.
- **Clarity beats cleverness.** If two designs compete, the one that's easier to understand wins.
- **Speed is a feature.** The interface moves with the user's intent — never gets in the way.

## Visual Identity

### Color Palette
| Name | Hex | Usage | Mood |
|------|-----|-------|------|
| Primary Blue | `#0D66CC` | Core actions, brand presence | Trust, stability |
| Secondary Blue | `#418BFF` | Interactive states, hover | Friendliness, energy |
| Accent Orange | `#FF6B35` | Calls to action, highlights | Warmth, confidence |
| Neutral Dark | `#0F172A` | Text, borders, deep backgrounds | Clarity, seriousness |
| Neutral Light | `#F8FAFC` | Backgrounds, subtle dividers | Breathing room, calm |

### Typography
- **Headings:** Poppins, 700 weight, 36px (H1) → 18px (H4), line-height 1.2
- **Body Copy:** Poppins, 400 weight, 16px, line-height 1.6
- **UI Labels:** Poppins, 500 weight, 14px, 0.5px letter spacing
- **Code:** Source Code Pro, 400 weight, 12px

### Spacing & Layout
- Base unit: 8px (ensures rhythm across all components)
- Margins: 8px, 16px, 24px, 32px, 48px
- Padding (components): 12px (compact), 16px (default), 20px (spacious)
- Gap (flex containers): 16px horizontal, 24px vertical (sections)

### Component Style
- Border radius: 8px (default), 12px (large components), 4px (tight UI)
- Button padding: 10px horizontal × 8px vertical (compact button)
- Card shadow: 0 2px 8px rgba(15, 23, 42, 0.1)
- Hover transition: 150ms ease-out (snappy, not sluggish)

## Imagery & Visual Language
- **Photography style:** Real people in real contexts (not stock). Natural lighting. Authentic moments.
- **Illustration:** Geometric, minimal, always purposeful. Use sparingly.
- **Motion:** Subtle. The user drives the animation; we don't announce it.
- **Patterns:** Avoid patterns. Let content breathe.

## Voice Characteristics
When a user interacts with Dynamous, they feel:
- **Capable.** The system believes they can do it.
- **Respected.** We don't talk down or over-explain.
- **Supported.** Help is one click away, never intrusive.

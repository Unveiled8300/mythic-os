# PPTX Generator: Layout Cookbook

A reference guide for all available slide layouts in the PPTX Generator. Use these templates to build consistent presentations aligned with your brand.

---

## Core Layouts (16 templates)

### 1. **Title Slide**
- **Purpose:** Opening slide for the presentation
- **Elements:** Title (H1), subtitle, date, branding
- **Brand integration:** Primary color background, logo placement
- **Example:** `"Let's build faster together" — Dynamous`

### 2. **Content Slide**
- **Purpose:** Main text + bullet points
- **Elements:** Title (H2), body copy (2–4 bullet points), side accent
- **Brand integration:** Primary color accent bar on left or right
- **Max content:** 4 bullets × 2 lines each

### 3. **Stats Slide**
- **Purpose:** Highlight key metrics or numbers
- **Elements:** Large number (80px+), label text, supporting stat
- **Brand integration:** Primary color for emphasis, accent color for secondary
- **Example:** `"23% faster" + "Teams using checklists"`

### 4. **Two-Column Layout**
- **Purpose:** Side-by-side comparison or content
- **Elements:** Title, two columns of equal width, icons or images optional
- **Brand integration:** Neutral divider line, color-coded column headers
- **Use case:** Before/After, Feature A vs Feature B

### 5. **Multi-Card Layout**
- **Purpose:** Display 3–4 distinct items (features, benefits, steps)
- **Elements:** Card title, icon or image, description (2–3 lines)
- **Brand integration:** Subtle shadow, accent color border
- **Grid:** 2×2 or 1×3

### 6. **Floating Cards**
- **Purpose:** Overlapping card design for visual interest
- **Elements:** Cards at different depths, connected by lines
- **Brand integration:** Each card uses secondary or accent color
- **Use case:** Process flow, staggered timeline

### 7. **Circular Hero**
- **Purpose:** Feature one image or icon prominently
- **Elements:** Large circular image (center), title below, description
- **Brand integration:** Circle border in primary color
- **Max text:** Title (1 line) + description (2 lines)

### 8. **Quote Slide**
- **Purpose:** Testimonial or powerful statement
- **Elements:** Quote text (large, italic), attribution, source
- **Brand integration:** Primary color accent (left border or quote mark)
- **Font:** Serif secondary font for elegance

### 9. **Chart/Graph Slide**
- **Purpose:** Display data visualization
- **Elements:** Title, chart, legend, supporting text
- **Brand integration:** Chart colors use brand palette (primary, accent, neutral)
- **Type:** Bar, line, pie, area charts

### 10. **Code Slide**
- **Purpose:** Display code snippet or technical content
- **Elements:** Title, monospace code block (2–10 lines), language badge
- **Brand integration:** Dark background (neutral_dark), code syntax highlighting
- **Fonts:** Mono font (Source Code Pro), 12–14px

### 11. **Image + Text**
- **Purpose:** Full-width image with text overlay or beside
- **Elements:** Image (4:3 or 16:9), title, description (1–3 lines)
- **Brand integration:** Text box with 0.5 opacity primary color background
- **Positioning:** Image left + text right, or text overlay bottom

### 12. **Section Divider**
- **Purpose:** Separate presentation into chapters/sections
- **Elements:** Section title (H1), optional subtitle
- **Brand integration:** Full-bleed accent color background, centered text
- **Use case:** "Part 1: Discovery" / "Part 2: Implementation"

### 13. **Timeline Slide**
- **Purpose:** Show chronological progression
- **Elements:** Horizontal or vertical timeline, milestones, labels
- **Brand integration:** Primary color for active milestone, neutral for completed
- **Milestone count:** 4–6 per slide

### 14. **Call-to-Action Slide**
- **Purpose:** Closing slide with invitation to act
- **Elements:** Large heading, brief supporting text, CTA button
- **Brand integration:** Button in accent color, brand colors
- **Button text:** "Get Started", "Learn More", "Join Us"

### 15. **Agenda/Outline Slide**
- **Purpose:** Show what's coming in the presentation
- **Elements:** Numbered list (3–5 items), optional icons
- **Brand integration:** Number circles in primary color
- **Max items:** 5 (keep it scannable)

### 16. **Thank You / End Slide**
- **Purpose:** Closing slide
- **Elements:** "Thank You", contact info, social links, logo
- **Brand integration:** Primary color background, white text
- **Contact:** Email, website, social handles

## LinkedIn Carousel Layouts (1:1 Format, 5 templates)

Carousels are exported as PDF at 1080×1080px (square).

### L1: **Text-Only Carousel**
- **Purpose:** Share a quote or key takeaway
- **Elements:** Single large quote (20–30 words), branding
- **Layout:** Centered text, 20% margin on all sides

### L2: **Text + Icon**
- **Purpose:** Highlight a single concept with visual
- **Elements:** Icon (256×256), title (2 lines), description (2 lines)
- **Layout:** Icon top (centered), text below

### L3: **List Carousel**
- **Purpose:** Multi-slide carousel with numbered items
- **Elements:** Number badge, title (1 line), description (2 lines)
- **Layout:** Number left, text right
- **Example:** "3 Ways to Ship Faster" (3 slides total)

### L4: **Before/After**
- **Purpose:** Show transformation or improvement
- **Elements:** Before state (left), arrow, After state (right)
- **Layout:** 50/50 split with divider
- **Use case:** "Before: Manual deployment" / "After: 1-click ship"

### L5: **Data Visualization**
- **Purpose:** Share a stat or research finding
- **Elements:** Large number (primary color, 72px+), label, source
- **Layout:** Number centered top (60%), label bottom (40%)
- **Example:** "23% faster deployment with our tool"

## Layout Selection Guide

| Goal | Best Layout | Why |
|------|-------------|-----|
| Introduce a concept | **Circular Hero** or **Title** | Strong visual hierarchy |
| Compare two things | **Two-Column** or **Before/After** | Clear side-by-side |
| Show a list | **Content Slide** or **Multi-Card** | Organized, scannable |
| Share a stat | **Stats Slide** | Numbers stand out |
| Feature an image | **Image + Text** | Full-bleed visual impact |
| Code demo | **Code Slide** | Proper syntax, readable |
| User testimonial | **Quote Slide** | Elegant, credible |
| Transition sections | **Section Divider** | Clear narrative break |
| End presentation | **Call-to-Action** or **Thank You** | Clear next step |
| LinkedIn sharing | **Carousel L1–L5** | Platform optimized |

## Customization Tips

1. **Spacing:** Use brand spacing units (8px grid). Don't crowd content.
2. **Colors:** Limit to 3 colors per slide (primary + accent + neutral).
3. **Fonts:** Stick to the 3 declared fonts (Poppins, Merriweather, Source Code Pro).
4. **Images:** 4:3 or 16:9 aspect ratio. Use real photography when possible.
5. **Icons:** Geometric, minimal style (no complex illustrations).
6. **Motion:** Subtle transitions only (fade, slide, 150ms duration).

## Exporting

### Standard Presentation
- Format: `.pptx` (PowerPoint)
- Aspect ratio: 16:9
- Resolution: 1920×1080px (72 DPI)
- Compatible with: PowerPoint, Google Slides, Keynote

### LinkedIn Carousel
- Format: `.pdf` (for upload)
- Aspect ratio: 1:1 (square)
- Resolution: 1080×1080px (72 DPI)
- File size: < 10MB per PDF

# Marketing Manager — Quick Reference
> Loaded when: brand, voice, SEO, metadata keywords detected.

## Business Model: B2B vs B2C
- B2C: emotional/convenience trigger, action-forward CTAs, consumer keywords, conversational tone
- B2B: ROI/authority trigger, value-proof CTAs, long-tail technical queries, authoritative tone

## User Persona (deliver before SPEC.md Section 3)
- Role, Technical Level, Pain Point, Desired Gain, Key Objection, Decision Trigger
- Key Objection must appear in at least one SPEC.md Section 6 criterion

## 7 Required SEO Tags (every HTML page)
1. `<title>` (≤60 chars, primary keyword near front)
2. `<meta name="description">` (≤160 chars, conversion-focused)
3. `<meta property="og:title">` (≤60 chars)
4. `<meta property="og:description">` (≤160 chars)
5. `<meta property="og:image">` (1200x630px min)
6. `<meta property="og:url">` (canonical, no trailing slashes)
7. `<link rel="canonical">` (prevents duplicate-content penalties)
